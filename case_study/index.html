Research Methodology

  1. Seed each app from apps.csv and discover official domains.
  2. Collect evidence pages from developer docs, API docs, auth docs, OAuth docs, and GraphQL docs.
  3. Classify fields with AI + deterministic heuristics.
  4. When key fields are Unknown, run a second-pass doc search before finalizing output.
  5. Write structured records to results.csv.
